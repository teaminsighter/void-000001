# Dashboard Pages Guide

The dashboard has 8 pages. Here's what each one does:

## 1. Home (/)

**Purpose:** Overview of everything at a glance.

**What you see:**
- Greeting with date
- Quick action buttons
- Stats cards (tasks, vault notes, mail, agent status)
- Today's tasks widget
- Inbox preview widget
- Recent notes widget
- Void-Haki search widget

**Actions:**
- Click quick actions to jump to features
- Toggle task checkboxes
- Click widgets to go to full pages

---

## 2. Agent (/agent)

**Purpose:** Chat with your AI assistant.

**What you see:**
- Chat message history
- Quick prompt buttons
- Text input field

**How to use:**
```
"Plan my day, I have 4 hours for work"
"Search my vault for project deadlines"
"Log: finished the marketing review"
"Remember that I prefer morning meetings"
"What are my goals for this month?"
```

**What Agent can do:**
- Read your vault for context
- Create daily plans
- Log entries to daily notes
- Search your notes semantically
- Remember things you tell it

---

## 3. Planner (/planner)

**Purpose:** Daily planning and task management.

**What you see:**
- Today's schedule (time blocks)
- Task list with checkboxes
- "Plan My Day" button

**How to use:**
1. Click "Plan My Day" to generate AI schedule
2. Check off tasks as you complete them
3. Tasks sync with your daily note

---

## 4. Vault (/vault)

**Purpose:** Browse all your notes.

**What you see:**
- File count and folder count
- Search bar (filename search)
- Folder filter pills
- File table (name, folder, modified, size)
- Semantic search section

**How to use:**
1. Click folder pills to filter by folder
2. Type in search to find files by name
3. Use "Ask Void-Haki" for semantic search
4. Click a file to view/edit (coming soon)

---

## 5. Mail (/mail)

**Purpose:** Email management (coming soon).

**Current status:** Shows "Email not connected"
**Future:** Will connect to Gmail for reading/sending emails

---

## 6. Research (/research)

**Purpose:** Deep semantic search.

**What you see:**
- Search input
- Vault results column (Void-Haki)
- Web results column (coming soon)
- Search history

**How to use:**
1. Type a question or topic
2. Click search or press Enter
3. See relevant results from your vault
4. History tracks your recent searches

---

## 7. Saved (/saved)

**Purpose:** Bookmarks and saved items (coming soon).

**Future:** Save articles, URLs, references

---

## 8. Bots (/bots)

**Purpose:** View automation status (coming soon).

**Future:** See all n8n workflows, their status, trigger them manually

---
