# Daily Workflow Guide

How to use VOID throughout your day for maximum productivity.

## Morning Routine (5 minutes)

### Option A: From Telegram
```
/plan 4h work today, meeting at 2pm, focus on marketing
```

### Option B: From Dashboard
1. Go to Agent page
2. Type: "Plan my day, I have 4 hours for work"
3. Agent creates your schedule

**Result:** Daily note created with time-blocked plan

---

## During the Day

### Quick Logging (Telegram)
Throughout the day, log what you're doing:
```
/log started deep work on proposal
/log finished first draft
/log call with Ahmed - he approved the design
/log break - getting coffee
/log back to work
/log completed proposal, sent to client
```

### Save Important Info (Telegram)
When you learn something the Agent should remember:
```
/memory context: Client deadline moved to March 1st
/memory preference: I focus better with music
/memory goal: Add 3 new features this week
```

### Deep Work (Dashboard)
When you need to:
- Search your vault → Research page
- Chat with Agent → Agent page
- Review notes → Vault page

---

## Evening Review (5 minutes)

### In Obsidian:
1. Open today's daily note: `01-Daily/2026-02-05.md`
2. Review your logs
3. Add any notes under `## Notes`
4. Check off completed tasks

### Optional: Plan Tomorrow
```
/plan tomorrow: 3h work, dentist at 11am
```

---

## Weekly Review (Sunday, 15 minutes)

### 1. Review the Week
- Open each daily note from the week
- What did you accomplish?
- What didn't get done?

### 2. Update Goals
In Obsidian, open `07-Agent-Memory/goals.md`:
- Check off completed goals
- Add new goals
- Adjust priorities

### 3. Process Inbox
- Move files from `00-Inbox/` to proper folders
- Keep Inbox clean

### 4. Plan Next Week
```
/memory context: This week focus on launching beta
```

## Workflows Cheat Sheet

```
QUICK LOG:
  Telegram → /log [text]
  → Saves to daily note

SAVE TO MEMORY:
  Telegram → /memory goal: [text]
  Telegram → /memory preference: [text]
  Telegram → /memory context: [text]
  → Updates Agent Memory files

PLAN DAY:
  Telegram → /plan [context]
  Dashboard → Agent → "Plan my day"
  → Creates daily note with schedule

SEARCH NOTES:
  Dashboard → Research → Type query
  Dashboard → Agent → "Search for [topic]"
  → Returns relevant vault results

EDIT NOTES:
  Obsidian → Open file → Edit
  → Changes sync everywhere
```

---

## Tips for Success

1. **Log often** - Small entries build great records
2. **Plan every morning** - Start the day with clarity
3. **Review in Obsidian** - See the big picture
4. **Update memory** - Help the Agent help you
5. **Keep Inbox clean** - Process weekly
